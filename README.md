# Sugarizer Server

Repository for Sugarizer Server Backend.

## Features

This project require npm 4+.

## Installation

Run following command in `terminal`.

>npm install

>npm test

## Running the server

Run following command in `terminal`.

>$ node sugarizer.js

Then navigate to `http://localhost:8080`
